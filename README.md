# PIDAL: Ride with Ease, Pedal with Peace🚲
**PIDAL** is an innovative campus mobility solution developed by First-Year Engineering students. It transforms underutilized university bicycle fleets into a secure, affordable, and eco-friendly transit network tailored for the campus community.

---

## 🌟 Vision & Impact
PIDAL addresses campus transit challenges by providing a seamless, "frictionless" experience for students. By leveraging technology to manage shared resources, we promote:
- **Sustainability:** Reducing the campus carbon footprint.
- **Efficiency:** Saving students time between far-flung lecture halls.
- **Accountability:** Ensuring bike security through strict community verification.

---

## 🛠 Key Features

### 🔍 Smart Access & Navigation
- **Live In-App Map:** Real-time visualization of bike stations and fleet availability using Google Maps API.
- **QR-Code Unlocking:** A high-speed scanning system (powered by CameraX and ML Kit) that verifies bike IDs against our Firebase backend for instant access.

### 💳 Financial & Security Systems
- **Credit-Based Payments:** A prepaid "top-up" system allowing students to manage ride balances without needing physical cash.
- **Strict User Verification:** Access is restricted exclusively to verified campus members to ensure equipment accountability.
- **Emergency Support:** A dedicated built-in hotline providing immediate support for rider safety or mechanical issues.

### 🎨 Modern UI/UX

- **"Eclipsed" Visual Identity:** The app utilizes a high-contrast **Purple and Yellow** theme designed to balance innovation with safety:
    - **Purple:** Represents creativity, uniqueness, and the "Eclipsed" brand innovation.
    - **Yellow:** Ensures high visibility and optimism, prioritizing rider safety and a friendly, approachable atmosphere for the campus community.
- **Glassmorphism Design:** A sophisticated UI style utilizing semi-transparent backgrounds and multi-layered gradients to create depth and a modern, high-tech aesthetic.
- **Material 3 Dashboard:** An intuitive user interface for quick navigation and real-time credit tracking.
- **Custom Iconography:** Specialized assets including the `qr_box.png` scanner overlay and custom `upload` action icons designed to maintain visual consistency.
---
## 🏗 Technical Architecture

- **Language:** Kotlin / Java
- **UI Framework:** XML with Material Design 3
- **Backend:** Firebase (Realtime Database & Authentication)
- **Scanning:** Google ML Kit (Barcode Scanning)
- **Camera:** Android CameraX Jetpack library
- **Mapping:** Google Maps SDK for Android
